## Adventure Templates

### The Brave Explorer

**Ages:** 7-10
**Pattern:** Branching
**Duration:** ~20 minutes
**Educational themes:** Courage, problem-solving, perseverance

**Synopsis:**
A young explorer discovers an old map in their grandmother's attic. The map leads to three different locations, each with its own challenge. The child must choose their path and overcome obstacles using creativity and courage.

**Key characters:**
- Protagonist (customizable name)
- Wise grandmother
- Friendly guide creature (owl/fox/turtle)

**Structure:**
```
Cover -> Finding the Map -> Grandmother's Story
  -> Choice: Forest Path | Mountain Trail | River Route
     -> Each path has unique challenge
        -> Resolution -> Treasure Discovery -> Home
```

---

### The Space Voyage

**Ages:** 6-9
**Pattern:** Linear with random encounters
**Duration:** ~15 minutes
**Educational themes:** Curiosity, teamwork, scientific thinking

**Synopsis:**
A child becomes an astronaut for a day and visits different planets. At each planet, they meet friendly aliens who teach them something new.

**Key characters:**

**Structure:**
```
Cover -> Rocket Launch -> Space Travel

## Bedtime Templates

### The Sleepy Star

**Ages:** 5-7
**Pattern:** Linear (autoplay)
**Duration:** ~10 minutes
**Educational themes:** Calm, gratitude, imagination

**Synopsis:**
A little star is too excited to fall asleep. The Moon sends them on a gentle journey across the night sky, visiting cloud friends and meeting sleepy animals, until the star finally feels peaceful and ready to rest.

**Key characters:**
- Little Star
- Wise Moon
- Cloud friends
- Sleepy animals

**Structure:**
```
Cover -> Star Can't Sleep -> Moon's Advice
  -> Floating Through Clouds -> Meeting Owl
  -> Counting Sheep -> Peaceful Rest
```

**Special notes:**
- Autoplay enabled
- Soft, slow narration
- Soothing sound descriptions
- Repetitive, calming phrases

---

### The Dream Garden

**Ages:** 5-8
**Pattern:** Hub (repeatable stories)
**Duration:** ~12 minutes (per story)
**Educational themes:** Relaxation, positive imagery, emotional comfort

**Synopsis:**
A magical garden where different flowers tell short, soothing stories. Child can choose which flower to visit, and each flower tells a different calming tale.

**Key characters:**
- Garden Keeper
- Rose (tells love stories)
- Sunflower (tells happy stories)
- Lavender (tells peaceful stories)

**Structure:**
```
Cover -> Garden Entrance -> Flower Selection (Hub)
  -> Rose's Story -> (return to hub)
  -> Sunflower's Story -> (return to hub)
  -> Lavender's Story -> (return to hub)
  -> Time to Sleep (exit)
```

## Educational Templates

### The Counting Quest

**Ages:** 5-6
**Pattern:** Interactive (quiz elements)
**Duration:** ~15 minutes
**Educational themes:** Numbers 1-10, counting, basic math

**Synopsis:**
A friendly dragon has lost their treasure and needs help counting to find it. The child helps count objects at each stop of the journey.

**Key characters:**
- Counting Dragon
- Number friends (1-10)

**Structure:**
```
Cover -> Dragon Needs Help
  -> Count 1-3 (objects) -> Reward
  -> Count 4-6 (objects) -> Reward
  -> Count 7-10 (objects) -> Treasure Found!
```

**Special notes:**
- Pause for child participation
- Celebrate correct answers
- Gentle encouragement for all attempts

---

### The Color Kingdom

**Ages:** 5-7
**Pattern:** Branching
**Duration:** ~12 minutes
**Educational themes:** Colors, observation, creativity

**Synopsis:**
In a kingdom where colors have faded, the child helps restore each color by finding objects of that color in their environment.

**Key characters:**
- Color Fairy
- Red Knight, Blue Princess, Yellow Sun, etc.

**Structure:**
```
Cover -> Colorless Kingdom -> Meet Color Fairy
  -> Red Zone -> Find red things
  -> Blue Zone -> Find blue things
  -> Yellow Zone -> Find yellow things
  -> Kingdom Restored!
```

**Ages:** 6-8
**Pattern:** Branching
**Duration:** ~15 minutes
**Educational themes:** Kindness, inclusion, empathy

**Synopsis:**
A new animal moves to the forest and feels lonely. The child's character befriends them and helps them meet other forest animals.

**Key characters:**
- Forest child (customizable)

- Other forest animals

**Structure:**

```

## Fantasy Templates

### The Magic Door

**Ages:** 7-10
**Pattern:** Branching with multiple endings
**Duration:** ~25 minutes
**Educational themes:** Imagination, choices matter, adventure

**Synopsis:**
A child discovers a magic door in an unexpected place. The door leads to different magical worlds based on their choices.

**Key characters:**
- Adventurous child (customizable)
- Door Keeper (mysterious guide)
- World-specific characters

**Structure:**
```
Cover -> Finding the Door -> Door Keeper's Offer
  -> Choice: Fairy World | Dinosaur World | Underwater World
     -> Each world has its own adventure
        -> Challenge -> Resolution
  -> Returning Home with New Wisdom
```

### The Dragon's Egg

**Ages:** 6-9
**Pattern:** Linear with care elements
**Duration:** ~18 minutes
**Educational themes:** Responsibility, nurturing, patience

**Synopsis:**
A child finds a mysterious egg and must care for it until it hatches. Different care choices affect the baby dragon's personality.

**Key characters:**
- Caring child (customizable)

**Structure:**
```
Cover -> Finding the Egg -> Taking It Home
  -> Day 1: Keeping Warm -> Day 2: Feeding Signs
  -> Day 3: Cracking! -> Hatching!

```

## Using Templates

When you start `/kidstory new`, you can:

1. **Choose a template:** Say "Use the Brave Explorer template"
2. **Modify a template:** "Use Sleepy Star but make it about a moon instead"
3. **Mix templates:** "Combine Forest Friends with educational counting"
4. **Start fresh:** "I have my own idea" for full custom creation

Templates provide:
- Proven story structures
- Age-appropriate pacing
- Character archetypes
- Educational tie-ins
- Flexible customization points
